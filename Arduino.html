What is Arduino?

Arduino is an open-source electronics platform that enables you to create interactive electronic projects. It consists of two main components: the Arduino board (the hardware) and the Arduino software (the programming environment).

 

Arduino is known for its beginner-friendliness due to several factors:

  • Simplicity: The Arduino programming language is relatively easy to understand, especially for those with little or no programming experience.
  • Abundance of Resources: Arduino has a large and supportive community that provides tutorials, example projects, and forums where you can seek guidance and inspiration.
  • Affordability: Arduino boards are relatively inexpensive, making them accessible to a wide range of users.
  • Versatility: Arduino can be used for various applications, from simple projects like blinking an LED to more complex projects involving robotics, home automation, or Internet of Things (IoT) devices.

Open source hardware
Arduino Board: The Arduino board is a compact and programmable circuit board that integrates a microcontroller as its central component. The microcontroller functions as a miniature computer capable of executing specific tasks based on programmed instructions. Arduino boards are available in diverse forms and dimensions, while consistently featuring input/output pins that facilitate connectivity and control of various electronic components, including but not limited to sensors, motors, LEDs, and other peripherals. These input/output pins serve as interfaces through which the Arduino board can interact with external devices, enabling users to create interactive and customizable electronic systems.

Arduino Uno: Our team will be using Arduino Uno for our class project and assesment.

Arduino Uno is a microcontroller board based on the ATmega328P ( datasheet ). It has 14 digital input/output pins (of which 6 can be used as PWM outputs), 6 analog inputs, a 16 MHz ceramic resonator (CSTCE16M0V53-R0), a USB connection, a power jack, an ICSP header and a reset button. It contains everything needed to support the microcontroller; simply connect it to a computer with a USB cable or power it with a AC-to-DC adapter or battery to get started.. You can tinker with your Uno without worrying too much about doing something wrong, worst case scenario you can replace the chip for a few dollars and start over again.

"Uno" means one in Italian and was chosen to mark the release of Arduino Software (IDE) 1.0. The Uno board and version 1.0 of Arduino Software (IDE) were the reference versions of Arduino, now evolved to newer releases. The Uno board is the first in a series of USB Arduino boards, and the reference model for the Arduino platform; for an extensive list of current, past or outdated boards see the Arduino index of boards..

Arduino IDE
Arduino software: To program the Arduino board, you use the Arduino software, also known as the Arduino IDE (Integrated Development Environment). The IDE provides a user-friendly interface where you can write, compile, and upload code to the Arduino board. The software, is a cross-platform application written in Java. It was originally designed to introduce programming to beginners, but it also has features that cater to more experienced programmers.
You can download Arduino Software from here according to your PC's version and then install by youself.

IDE interface:

  • Menu Bar (file menu, edit menu, program menu, tools menu, and help menu): The menu bar at the top of the Arduino IDE program provides access to various features and settings. It includes options for managing sketches and selecting the Arduino board and port.
  • Toolbar: The toolbar contains various buttons for common actions. Verify, compile, upload, debug, new sketch, open sketch, save sketch, serial plotter and serial monitor
  • Sketch Area: This is the main area where you write your Arduino code. It is a text editor that supports syntax highlighting and auto-completion.
  • Status Bar: The status bar at the bottom of the Arduino IDE program displays useful information, such as the current board and port selection, the status of the compilation and upload process, and any error messages.

Configure IDE and hardware connect:

  • Connect your hardware device: Connect your Arduino board to your computer using a USB cable.
  • Open Arduino IDE: Launch the Arduino IDE program on your computer. You should see the main interface with the Sketch Area, Toolbar, Menu Bar, and other interface parts.
  • Configure Board: In the Arduino IDE, go to the "Tools" menu and select the appropriate board that you are using from the "Board" submenu. This selection ensures that the IDE compiles the code correctly for your specific board. If your board is not listed, you may need to install board support packages (if available) or manually configure the board settings.
  • Select Port: After selecting the board, go to the "Tools" menu again and choose from the "Port" submenu the appropriate serial port to which your Arduino board is connected. This selection allows the IDE to communicate with your board for uploading code and accessing the serial monitor.
  • Monitor Serial Output (optional): If your code includes sending and receiving data via the serial port, you can open the Serial Monitor in the Arduino IDE by clicking on the magnifying glass icon in the toolbar. This tool allows you to view the data being sent and received by the Arduino board, which can be helpful for debugging and monitoring purposes.
Now you are you are ready to write your code in the Sketch Area of the Arduino IDE, to control or interact with your connected hardware. The IDE provides a user-friendly text editor with features like syntax highlighting and auto-completion to assist you in writing code.
Coding

The Arduino coding language is based on C/C++ and is designed to be beginner-friendly and easy to understand.

 

Basic Structure: The structure consists of two main functions:

  • setup(): The setup() function is called when a sketch starts. Use it to initialize variables, pin modes, start using libraries, etc. The setup function will only run once, after each powerup or reset of the board.
  • loop(): After creating a setup() function, the loop() function does precisely what its name suggests, and loops consecutively, allowing your program to change and respond as it runs. Code in the loop() section of your sketch is used to actively control the board.
Reference: https://docs.arduino.cc/built-in-examples/basics/BareMinimum
Digital and Analog I/O: Arduino boards have digital and analog input/output pins that allow you to interact with the physical world. The Arduino language provides functions to read from and write to these pins. For example, you can use digitalRead() to read the state of a digital pin or analogWrite() to output a PWM (Pulse Width Modulation) signal.
Compile and upload the code
Once you have written your code, click on the "Verify" button (checkmark icon) in the toolbar to compile your code. This step ensures that there are no syntax errors in your code. If the verification is successful, you can proceed to upload the code to your Arduino board by clicking on the "Upload" button (right arrow icon) in the toolbar.
Once the upload is finished, the code will start running on the Arduino board
