Power and speed test
In this Test we use 2mm transparent acrylic.
This demo will be prepared to verify the parameters of speed/power at which the laser either completely cuts or does not cut the material provided.
We designed a total of 9 sets, each with nine different power levels (90%, 80%, 70%, 60%, 50%, 40%, 30%, 20%, 10%). Each set contains six cuts at different speeds. The speeds are 10 mm/s, 20 mm/s, 30 mm/s, 40 mm/s, 50 mm/s, and 60 mm/s.
For this test, the design was prepared in advance using Adobe Illustrator. Other programs such as AutoCAD and CorelDraw can also be used.
Follow this link to download and installing Adobe Illustrator: https://www.adobe.com/hk_en/products/illustrator/free-trial-download.html
Once in the open interface of the LaserCAD software, we need to create a new blank workspace.
Next, we click on the File menu and look for the import option
In the new open window, we will search for the file that we have prepared

Once imported, we will use the Engrave parameter to ensure that all the markings for each Power=# that identifies each sets are correctly marked on the surface of the material.

  • Select each text using the selection tool.
  • Choose a color.
  • Double-click on the color that has now been added in the Layer Options panel.
  • In the new window, select Work Mode: Engrave and press the Ok button.

Let’s proceed with adjusting colors for each layer and corresponding cutting parameters for each module.

 

Let’s walk through the steps for the external cutting layer of the P=10 module:

 

  • Select a color of your preference to identify the layer and double-click on the chosen color in the Layer Options panel.
  • In the Work Mode section, select “Cut”.
  • Adjust the Maximum and Minimum Power to 10.
  • Set the Speed value to 60.

 

Repeat these steps for the interior cuts of the module, selecting a different color for each layer and adjusting the speed to 50, 40, 30, 20, and 10 as you move inward.

To deal with the colour limitation in the programme, we will separate the cutting module pairs into different files. In this way, we will be able to work with the 36 power per speed combinations without colour restrictions.

KERF TEST

The kerf test in laser cutting is a crucial step to ensure precise and accurate results.

What is Kerf?

  • Kerf refers to the width of material removed during the laser cutting process. It represents the gap created by the laser beam as it burns away material.
  • The actual kerf can vary based on factors such as material type, thickness, and laser cutter settings.
How we Perform a Kerf Test:
  • Start by cutting 10 rectangles of 5 mm each (you can use any other size) from 2mm acrylic material using the laser cutter.

We made two test modules varying the Power (90% and 80 %) and with the same speed value (20).

  • Measure the 10 rectangles together using calipers after cutting to determine the actual size.
