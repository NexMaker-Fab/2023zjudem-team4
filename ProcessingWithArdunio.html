TWO PLAYER - WASTE POP GAME

This project was created by using processing 4 (4.3) combined with the Arduino. This is a interactive two player game that helps to spread knowledge about waste management while playing a game.

 

Using Processing where the one of the players press keys ('R', 'G', or 'B') to match the letter with circle color. It utilizes serial communication to receive signals from an Arduino to change the circle's color. The game tracks points and missed, if you miss three in a raw you will loose the game. In the end you will receive a short message to influence the player to manage waste properly.

Components Needed:

Hardware:

Arduino Uno

Ultrasonic distance sensor

Regular LEDs

Resistors (330-500 ohms)

Breadboard

Jumper wires

 

Software:

Arduino

Processing

Wiring Instructions:

1. Connect Ultrasonic Sensor to Arduino:

VCC -> 5V on Arduino

Trig -> Digital Pin 2 on Arduino

Echo -> Digital Pin 3 on Arduino

GND -> GND on Arduino

2. Connect Regular LEDs to Arduino:

Connect the anode (+) of each LED to a digital pin on the Arduino through a resistor (330-500 ohms).

Connect the cathode (-) of each LED to GND on the Arduino.

Ensure to use appropriate resistors to limit current flow and protect the LEDs.

Use breadboard as the base to connect different components and jump wires to connect them.

Developing the Arduino Code

Variable Initialization:

  • unsigned long previousMillis = 0;

This variable is used to store the previously recorded time in milliseconds. It's initialized to 0 at the start of the program.

  • unsigned long currentMillis = 0;

Similar to previousMillis, this variable of type unsigned long is used to store the current time in milliseconds. It's also initialized to 0.

  • unsigned long elapsedTime = 0;

This variable is utilized to calculate the elapsed time between measurements. It's initialized to 0 at the beginning of the program.

  • long duration, distance;

These variables are used to hold the duration (the time it takes for the ultrasonic signal to travel and return) and the calculated distance in centimeters, respectively. They are declared but not initialized explicitly at the beginning because their values will be assigned later in the code based on the sensor readings.

unsigned long previousMillis = 0;

unsigned long currentMillis = 0;

unsigned long elapsedTime = 0;

Setup Function:

Initializes serial communication.

Sets the pin modes for the sensor trigger and echo pins as well as for the LED pins.

void setup() {

Serial.begin(9600);

pinMode(TRIGGER_PIN, OUTPUT);

pinMode(ECHO_PIN, INPUT);

pinMode(GREEN_LED_PIN, OUTPUT);

pinMode(RED_LED_PIN, OUTPUT);

pinMode(BLUE_LED_PIN, OUTPUT);

}

Loop Function:

Measures the distance using the ultrasonic sensor.

Calculates the elapsed time since the previous measurement.

Based on the elapsed time, it adjusts delays to change the frequency of sensor readings.

Prints the waste type detected on the Serial Monitor for debugging purposes.

void loop() {

long duration, distance;

digitalWrite(TRIGGER_PIN, LOW);

delayMicroseconds(2);

digitalWrite(TRIGGER_PIN, HIGH);

delayMicroseconds(10);

digitalWrite(TRIGGER_PIN, LOW);

duration = pulseIn(ECHO_PIN, HIGH);

distance = (duration / 2) / 29.1; // Calculate distance in centimeters

currentMillis = millis();

elapsedTime = currentMillis - previousMillis;

if (elapsedTime >= 0 && elapsedTime < 15000) { // 0-30 seconds

delay(2000); // 2 seconds delay

} else if (elapsedTime >= 15000 && elapsedTime < 30000) { // 30-60 seconds

delay(1000); // 1 second delay

} else if (elapsedTime >= 30000 && elapsedTime < 45000) { // 30-60 seconds

delay(500); // 1 second delay

} else if (elapsedTime >= 45000 && elapsedTime < 60000) { // 30-60 seconds

delay(250); // 1 second delay

} else if (elapsedTime >= 60000) { // 60 seconds and after

delay(100); // 0.5 second delay

}

if (distance < 10) {

activategreen(); // If object is close, activate green LED

Serial.println("RECYCLABLE");

} else if (distance >= 10 && distance < 20) {

activatered(); // If object is at medium distance, activate red LED

Serial.println("NON_RECYCLABLE");

} else if (distance >= 20 && distance < 30) {

activateblue(); // If object is at far distance, activate blue bin LED

Serial.println("COMPOSTABLE");

}

delay(100); // Adjust delay as needed for data transmission

}

Distance Measurement and LED Activation:

Determines the distance measured by the ultrasonic sensor and activates the appropriate LED based on the distance thresholds set (activategreen(), activatered(), activateblue()).

void activategreen() {

digitalWrite(GREEN_LED_PIN, HIGH); // Turn on green LED

digitalWrite(RED_LED_PIN, LOW); // Turn off red LED

digitalWrite(BLUE_LED_PIN, LOW); // Turn off blue LED

}

digitalWrite(GREEN_LED_PIN, LOW); // Turn off green LED

digitalWrite(RED_LED_PIN, HIGH); // Turn on red LED

digitalWrite(BLUE_LED_PIN, LOW); // Turn off blue LED

void activateblue() {

digitalWrite(GREEN_LED_PIN, LOW); // Turn off green LED

digitalWrite(RED_LED_PIN, LOW); // Turn off red LED

digitalWrite(BLUE_LED_PIN, HIGH); // Turn on blue LED

}

Developing the Processing Code

Serial Communication Setup and Variables

 

  • Serial Communication Setup :

import processing.serial.* - Imports the serial library to facilitate communication with external devices.

 

  • Variable Initialization :

myPort : Declares a variable for serial communication.

circleFillColor : Sets the default color for the circle.

colorKeys : Defines an array containing characters ('R', 'G', 'B') as keys for colors.

points : Tracks the player's points.

missedColors : Counts the number of missed colors.

gameStarted : Boolean flag to indicate if the game has begun.

circleSize : Initial size for the displayed circle.

import processing.serial.*;

Serial myPort; // Create a variable for the serial port

int circleFillColor = color(255); // Default fill color for the circle

char[] colorKeys = {'R', 'G', 'B'}; // Keys corresponding to colors

int points = 0; // Player's points

int missedColors = 0; // Counter for missed colors

boolean gameStarted = false; // Flag to start the game

Setup Function

setup() - Function

size(800, 600) - Sets the dimensions of the window to 800x600 pixels.

myPort = new Serial(this, "COM3", 9600) - Initializes serial communication with the specified port ("COM3") and baud rate (9600).

void setup() {

size(800, 600); // Set the size of the Processing window

myPort = new Serial(this, "COM3", 9600); // Replace "COM3" with your Arduino's port

}

Draw Function

 

draw() - Function

background(220) - Clears the window and sets the background color to light gray.

if (gameStarted) { ... } - Displays the circle if the game has started.

textAlign(CENTER) - Sets the text alignment to center for score, missed count, and instructions.

displayInfo1();, displayInfo2();, displayInstructions(); - Calls functions to display score, missed count, and game instructions.

if (missedColors >= 3) { gameOver(); } - Checks if the player has missed three colors and triggers the game over function.

while (myPort.available() > 0) { ... } - Reads data from the serial port.

wasteType = myPort.readStringUntil('\n'); - Reads a string from the serial port until a newline character is encountered.

Processes the received data to change the circle's color based on the signals ("GREEN" "RED" or "BLUE") received from an external device (Arduino).

void draw() {

background(220); // Set background color

if (gameStarted) {

// Display the circle based on the selected color

fill(circleFillColor);

ellipse(width / 2, height / 2, 100, 100); // Display a circle at the center of the window

}

// Display score and missed colors

textAlign(CENTER);

displayInfo1();

displayInfo2();

displayInstructions(); // Display game instructions

// Check for game over condition

if (missedColors >= 3) {

gameOver();

}

// Read data from the serial port

while (myPort.available() > 0) {

String wasteType = myPort.readStringUntil('\n'); // Read data from serial port

if (wasteType != null) {

wasteType = wasteType.trim(); // Remove whitespace

// Set circle color based on received distance signal

if (wasteType.equals("RECYCLABLE")) {

circleFillColor = color(0, 255, 0); // Green for recyclable

} else if (wasteType.equals("NON_RECYCLABLE")) {

circleFillColor = color(255, 0, 0); // Red for non-recyclable

} else if (wasteType.equals("COMPOSTABLE")) {

circleFillColor = color(0, 0, 255); // Blue for compostable

}

gameStarted = true; // Start the game when signal received

}

}

}

Key Press Handling

 

keyPressed() Function:

Handles keyboard inputs from the player.

Checks if the game has started and compares the pressed key to the expected color keys.

Updates points, missed colors, and triggers functions to change color and size or end the game based on player input.

void keyPressed() {

if (gameStarted) {

char keyPressed = Character.toUpperCase(key);

for (int i = 0; i < colorKeys.length; i++) {

if (keyPressed == colorKeys[i]) {

if (keyPressed == getColorChar(circleFillColor)) {

// If player presses the correct key corresponding to the displayed color

points++;

missedColors = 0; // Reset missed colors

gameStarted = false; // Set to false to refresh the color

pickRandomColor(); // Choose a new random color

} else {

missedColors++;

if (missedColors >= 3) {

gameOver();

}

}

break; // Exit loop after handling input

}

}

} else {

// Start the game when the ENTER key is pressed

if (keyCode == ENTER) {

gameStarted = true;

pickRandomColor(); // Choose a random color to start

}

}

}

Game Logic Functions

 

pickRandomColor():

Randomly selects a color ('R', 'G', or 'B') and changes the displayed circle's color accordingly.

getColorChar():

Determines the character corresponding to the current circle's color.

randomizeSize():

Randomly changes the size of the displayed circle within a defined range to add variation to the game.

gameOver():

Displays a "Game Over!" message and stops the game loop when the game ends due to missing colors.

void pickRandomColor() {

// Pick a random color and display the corresponding color

int randomIndex = int(random(colorKeys.length));

char selectedColor = colorKeys[randomIndex];

if (selectedColor == 'R') {

circleFillColor = color(255, 0, 0); // Red

} else if (selectedColor == 'G') {

circleFillColor = color(0, 255, 0); // Green

} else if (selectedColor == 'B') {

circleFillColor = color(0, 0, 255); // Blue

}

}

char getColorChar(int fillColor) {

// Get the character corresponding to the color

if (fillColor == color(0, 255, 0)) {

return 'G';

} else if (fillColor == color(255, 0, 0)) {

return 'R';

} else if (fillColor == color(0, 0, 255)) {

return 'B';

}

return ' ';

}

void gameOver() {

// Display game over message

fill(0, 0, 2);

textSize(80);

textAlign(CENTER);

text("Game Over!", width / 2, height / 2);

// Show information about waste recycling in a new window

showRecyclingInfo();

}

Capturing Simple User Interaction with Processing

It utilizes serial communication to receive signals from an Arduino to change the circle's color. The game tracks points, missed colors, and displays instructions while allowing the player to interact using keyboard inputs.
